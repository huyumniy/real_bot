1. Закидуємо в окрему папку файл server.js
2. Інсталюємо залежності через команди в терміналі (можна через VS code)
npm install express 
npm install body-parser
3. Запускаємо сервер командою node server.js

На різних серверах повинні бути різні порти прописані як у серверному скрипті так і користувацькому
